¿Quién inventó el sistema de control de versión Git y por qué?

El creador es Linus Torvalds el creado del sistema operativo Linux. Se creo para llevar un control de versiones y para poder trabajar en equipo pudiendo respaldar el proyecto.

¿A quién pertenece actualmente Github y por qué?

Le pertenece actualmente a Micrososft. Porque quisieron apostar mas fuerte por el Open Source.
