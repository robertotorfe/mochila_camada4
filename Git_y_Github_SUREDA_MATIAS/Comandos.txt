Listado de comandos

pwd 
me muestra en qué directorio me encuentro

cd 
es usado para moverme entre directorios

cd ..
retrocede un directorio, se puede stackear

cd ~
me lleva a mi usuario

cd _________ 
me mueve al directoriodel nombre que escribí

mkdir
make directory, crear una carpeta-directorio

ls
me muestra el contenido de mi directorio actual

clear 
limpia la pagina de la terminal en la que estoy

touch ________
crea un archivo, con el nombre puesto, se puede agregar extensión, txt, jpg, pdf

mv _origen/nombre_ _nuevo nombre_
para renombrar

mv _origen/nombre_ _destino_
mover

cp _origen/nombre_ _destino_
copiar

rm _origen/nombre_ _destino_
eliminar o remover

ls -R
muestra todos los archivos sobre la carpeta en la que estou, inluyendo subcarpetas

code .
abre vsc sobre la carpeta en la que estoy

git init
me permite crear un nuevo repositorio

git status
me permite ver el estado del repositorio

git clone _enlace de clonación_
clona un repositorio remoto a mi directorio

git branch _nombre_
crea una rama en mi repositorio

git checkout _nombre_
me permite moverme a la rama seleccionada

git add _nombre del archivo_
me permite añadir a mi repositorio el archivo nombrado

git add .
agrego todos los archivos al repositorio

git commit -m "___"
me permite crear un commit con los archivos en mi repositorio

git push origin _nombre de la rama_
me permite subir al repositorio remoto el commit actual
