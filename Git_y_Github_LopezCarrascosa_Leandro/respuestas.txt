a. ¿Quién inventó el sistema de control de versión Git y por qué?
Lo inventó Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005, para
mejorar la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de 
aplicaciones cuando estas tienen un gran número de archivos de código fuente.
