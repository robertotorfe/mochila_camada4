¿Quién inventó el sistema de control de versión Git y por qué?

Fue diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.


¿A quién pertenece actualmente Github y por qué?

¿Hay otra forma que no sea la terminal para trabajar con Github?

Se puede trabajar desde Github Desktop o bien desde la interfaz web ya sea generando cambios a los archivos desde el code web o bien subiendo archivos manualmente desde la adhesion de archivos.
